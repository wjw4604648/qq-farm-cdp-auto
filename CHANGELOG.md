# Changelog

以下仅保留当前版本仍成立的用户侧变更与能力说明。

## 2026-04-22

### 仓库页与出售链路

- 新增独立 `仓库` 页签，放在 `土地详情` 与 `运行时` 之间；仓库数据改为后台隐藏打开仓库后读取，并把四个仓库页签中的物品聚合为统一预览。
- 仓库预览新增本地缓存；首次进入仓库页或远端临时读取失败时，会优先回退展示最近一次成功读取到的仓库快照，避免页面出现整块空白。
- 仓库调度已接入后台定时刷新：默认每 `5` 分钟自动刷新一次仓库缓存；自动出售默认每 `12` 小时执行一次，若配置为 `0` 则每次刷新仓库时都尝试出售可预估物品。
- 手动出售与自动出售都已接入账户状态“运行统计”；除 `出售` 件数外，新增 `出售估值` 金币统计，并按天写入自动农场统计文件。
- 修复仓库聚合卡片出售时把总数量错误压到单个仓格导致“数量不足”的问题：现在会按同类物品命中多个真实仓格，保留真实 `uid` 后按批量出售链路提交。
- 修复仓库出售匹配与结果观察逻辑：统一仓库展示与卖出请求的物品 ID 解析规则，卖出后改为重新扫描四个仓库页签并聚合比对，减少 `warehouse_sell_no_match` 与 `warehouse_sell_result_not_observed` 误报。
- 优化仓库与自动农场运行时冲突：仓库刷新/出售会等待自动农场当前轮次结束后再执行，不再直接因“自动农场运行中”而失败。
- 优化刷新交互：手动刷新仓库与自动出售已合并为同一次仓库会话，刷新时会先扫四个页签，再按需要直接复用当前已打开的仓库执行自动出售，减少重复开关仓库。

## 2026-04-21

### 账户状态统计持久化

- 账户状态页中的自动农场“今日统计”已改为本地按天持久化，不再只存在内存里；网关或项目重启后会自动恢复当天统计，不会被清空。
- 新增统计目录 `data/account-stats/auto-farm-daily/`，按 `YYYY-MM-DD.json` 保存每日运行统计，便于后续排查与归档。
- 统计会按北京时间（UTC+8）自动隔日切换到新文件，并自动保留最近 `30` 天历史，超出的旧文件会自动清理。
- 自动农场管理器与网关已接入这套每日统计存储：启动时加载最近 30 天记录，运行过程中每轮自动写回当天统计。

### 账户状态页统计增强

- 账户状态页中的“今日统计”卡片升级为“运行统计”，支持直接切换查看 `今天 / 近三天 / 近七天 / 近十五天 / 近三十天` 的汇总数据。
- 新增 `指定日期` 下拉，可单独查看最近 30 天内任意一天的统计明细；切回区间视图时无需刷新页面。
- 统计卡额外补充了查看范围、覆盖天数、总运行轮次和最近更新时间，让多日数据更容易对照。

### 调度中心与串行执行

- 控制页新增独立 `调度中心` 页签，放在“自动农场”旁边，用来统一管理常驻任务的全局执行顺序。
- 新增前端串行调度器配置：支持总开关、任务间缓冲时间、任务启停、优先级、轮询间隔和运行模式，并支持一键上移/下移优先度与恢复推荐值。
- 调度中心会显示当前运行队列、下一次预计执行时间和最近一次结果，用来降低多个页面任务并发轮询导致的小程序卡顿或白屏风险。

### 自动农场细分开关

- 自动农场 `模块开关 -> 基础任务` 现在支持二级动作拆分：`收获 / 除草 / 浇水 / 杀虫` 四项可分别开启或关闭，默认仍全部开启。
- 基础任务二级动作已真正接入执行链，不是仅前端展示；配置会持久化到农场配置，并透传到自动农场执行器。
- 当四个基础动作全部关闭，且未配置自动种植或自动施肥时，自己农场本轮会直接跳过空跑，减少无效轮询与进场操作。
- 基础任务二级动作区域改为主卡片内的小折叠面板，避免原先平铺四张大卡片导致层级过重、界面突兀。

### 好友巡查调度拆分

- `调度与流程` 中的原单一“好友间隔”已拆为 `偷菜巡查间隔` 与 `帮助巡查间隔` 两个独立配置项，前端表单、配置持久化和自动农场状态卡片已同步更新。
- 自动农场管理器已拆分好友偷菜与好友帮忙的独立到期判断、上次执行时间和下次预计运行时间，不再共用同一套好友轮次间隔。
- 自动农场开始调度日志会明确区分“好友偷菜=执行/跳过”“好友帮忙=执行/跳过”，方便定位是哪一路好友巡查触发。

### 配置交互与资源

- 控制页新增全局“有修改未保存”浮动提示；自动农场和调度中心中只要有配置变更，就会在页面左下角显示保存提示，点击即可立即持久化生效。
- 未保存提示已覆盖普通表单改动、调度优先级移动、恢复默认调度、黑名单作物点选等非标准输入入口；保存成功后会自动消失，保存中会显示按钮状态。

## 2026-04-14

### QQ 弹窗拦截与启动修复

- 新增 QQ 奖励弹窗拦截模块：可在控制页“自动农场 -> 模块开关”中独立开启或关闭，运行时会自动识别并隐藏 `view_get_rewards` 奖励弹窗及其遮罩层。
- 奖励弹窗拦截状态改为独立运行时开关，不写入自动农场策略配置；控制页同步增加模块状态卡、运行提示和接口轮询。
- 修复奖励弹窗拦截关闭后仍继续生效的问题：底层拦截器拆分“启用态”和“执行态”，停止时会终止旧轮询，不再被执行中的旧定时器重新挂起。
- 精简奖励弹窗拦截展示：去掉误导性的“已处理次数 / 上次命中”显示，状态卡和提示语改为仅展示当前开关与检查状态。
- 修复微信路线启动时 `--wx` / `--runtime` 参数透传到 WMPF 后触发 `ERR_PARSE_ARGS_UNKNOWN_OPTION` 的问题；`run.cjs` 会先吸收外层运行时参数，`wmpf` CLI 也兼容忽略这些别名。

### 界面与展示

- 作物分析页改为卡片式展示，排序与筛选逻辑不变；策略推荐区同步改为简洁卡片风格，并保留“应用这个策略”快捷操作。
- 土地详情页释放地块卡片空间：成熟事件从主布局移至按钮触发的二级悬浮弹窗，支持遮罩点击和 `Esc` 关闭。
- 地块卡片移除重复季数展示，仅保留主信息区季数，避免视觉冗余。
- 运行时间显示优化：账户状态页补齐经验速率与预计升级时间展示，自动农场模块状态区突出下一次预计运行时间，便于观察调度节奏。

### 好友与经验

- 修复好友帮忙经验跨日未清零问题：按北京时间（UTC+8）切日，并在状态读取/调度/写入前主动执行跨日滚动重置。
- 调整好友帮忙停止逻辑：改为按北京时间统计“每日帮忙次数上限”，默认每天最多帮忙 30 个好友，达到上限后暂停当天帮忙并在次日自动恢复。

## 2026-04-13

### 启动与运行时

- 修复微信路线启动时 `--wx` 透传导致的注入报错。
- 新增 QQ 宿主自动重连与自动农场调用前恢复，网络异常后会自动尝试重连并重试一次。

### 种植与施肥

- 修复空地识别问题，自动种植现在只处理可交互空地，不再误判未解锁土地。
- 修复背包种子不足时误开商店卡住的问题；背包优先和排行策略会按现有库存部分种植，不再强制补购。
- 地块详情新增单地块手动施肥按钮，支持直接执行无机/有机施肥。
- 新增自动施肥策略，支持 `仅催熟时使用无机`、`仅催熟时使用有机`、`种植无机，催熟有机` 三种模式。
- 自动施肥支持成熟前阈值催熟；阈值可配置，默认 `300s`，多季作物的后续季也会在临近成熟时补肥。
- 自动施肥支持按土地品质筛选，以及重启后可恢复的施肥状态缓存 `data/auto-fertilizer-state.json`。
- 自动施肥会识别同一作物已使用过的同类肥料，自动跳过并记录原因，避免重复施肥。
- 停止自动化时会向当前轮任务下发中止信号，施肥和好友巡检不会继续拖尾执行。

### 好友与经验

- 自动农场新增好友帮忙模块，可按好友农场当前可浇水、除草、除虫状态执行帮忙。
- 好友操作列表补齐单个好友手动帮忙入口，并将帮忙结果写入统一日志。
- 新增好友帮忙状态缓存 `data/friend-help-exp-cache.json`，按北京时间跨日重置。
- 支持“每日帮忙上限”；达到当天配置的好友帮忙次数后，会暂停后续帮忙并在北京时间次日恢复。

### 账户、界面与日志

- 地块详情中的施肥按钮文案简化为 `无机`、`有机`。
- 清理了地块详情中的多余土地品质展示。
- 未适配到作物图标时，默认回退到 `400.jpg`。
- 修复账户状态经验值解析，等级进度、经验/小时和升级预计时间恢复正常。
- 账户状态页中的运行日志已拆分为独立的 `日志中心` 页签。
- 新增账户状态本地缓存 `data/player-profile-cache.json`，默认保留 24 小时。
- 网关重启、弱网或运行时未就绪时，账户状态、作物分析和种植策略会优先回退最近一次可用的等级与经验数据，避免等级读成 `0` 触发种植异常。
- 自动农场状态面板拆分显示各模块启用状态与下一次预计运行时间。
- 自动农场日志补充到地块级和好友级，收取、种植、施肥、帮忙会显示目标、结果和跳过原因。
- 页面左侧新增 GitHub 悬浮入口，弹窗展示开源说明、项目地址、挂机推荐链接和 QQ 群号。
